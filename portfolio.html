Graph ADT

Aggie Shell

This project was an exercise in implementing a Linux shell through C++. Features:

I recently completed a project that was an exercise in implementing a Linux shell using C++. This project showcases my proficiency in the C++ programming language and my understanding of operating system concepts. One of the key features of the shell was the ability to run Linux commands as child processes, enabling the shell to perform complex operations. The implementation of inter-process communication allowed for seamless communication between the shell and the child processes, resulting in efficient and accurate execution of commands. In addition, the shell provided the ability to navigate through the file system and perform file input/output redirection. This demonstrated my understanding of how to interact with the file system and manage data streams in a shell environment. The shell was designed to function almost as well as BASH, the standard shell on most Linux systems. The use of execvp() to run Linux commands that sit as executables in the system demonstrates my understanding of the underlying system calls used to execute commands. Overall, the Linux shell project is a testament to my technical abilities and my passion for programming. My ability to design, implement, and deliver a functional Linux shell that is capable of running Linux commands and providing a rich set of features highlights my skills in software development and problem-solving.
