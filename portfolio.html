Here are a few of the projects I have completed.





Graph ADT

This is a project about developing a Graph Abstract Data Type through C++. The project was developed in a team of two. It was an exercise in and Object Oriented Programming and Algorithm Analysis. The image on the right is pictoral representation of the data structure. Using Dijkstra's Algorithm, we were able to compute the shortest path between a chosen root vertex and every other vertex.









Aggie Shell

This project was an exercise in implementing a Linux shell through C++. Features:

  • Run Linux commands as child processes
  • Inter-process communication
  • Navigate through the file system
  • File input/out redirection

The shell has the ability to function almost as well as BASH. It runs Linux commands that sit as executables in the system through a a call to execvp().
